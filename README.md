# Wanderlust - Airbnb Clone

A full-stack property listing web application built with Node.js, Express.js, MongoDB, and EJS.

## Features

- Browse all property listings on the home page
- View detailed information for each individual listing
- Create new property listings via a form
- Edit and update existing listings
- Delete listings
- Server-side form validation using Joi schema
- Custom error handling with ExpressError and HTTP status codes
- Reusable asyncWrap utility to handle async errors without repetitive try-catch blocks
- Default image URL fallback when no image is provided
- Reusable EJS layout using ejs-mate
